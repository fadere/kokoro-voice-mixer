TTS Voice Blender
A Python-based automated pipeline for generating multi-speaker conversational audio. 

Features
Multi-Speaker Orchestration: Blends multiple voices naturally to create engaging conversational audio from raw text.

High-Fidelity Speech Synthesis: Leverages Kokoro TTS for high-quality, lifelike voice generation.

Automated Transcription: Integrates faster-whisper for rapid, accurate transcription and alignment.

Content Extraction: Built-in tools to automate the extraction and formatting of YouTube transcripts.

Hardware Optimized: Configured to run efficiently on Google Colab (T4 GPUs) or locally on Mac Mini unified memory clusters.

Prerequisites
Python 3.8+

Kokoro TTS

faster-whisper

yt-dlp or youtube-transcript-api (for transcript extraction)

FFmpeg (for audio processing and merging)
